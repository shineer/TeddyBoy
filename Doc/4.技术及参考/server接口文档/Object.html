g# 对象定义 #

1.0 平台信息对象 platformInfo

对象说明: 

处理客户端信息、手机信息、网络信息数据封装。

{
    "platformInfo":{
            "version":版本号,
            "fid":渠道号,
            "platform":平台(3->Android),
            "product":产品号(8->宝葫芦项目),
            "phonetype":手机型号,
            "pid":手机串号,
            "w":分辨率宽,
            "h":分辨率高,
            "systemVersion":系统版本,
            "netType":联网类型(0->无网络,2->wifi,3->cmwap,4->cmnet,5->ctnet,6->ctwap,7->3gwap,8->3gnet,9->uniwap,10->uninet)
            "imsi":sim卡imsi号,
            "mobileIP":手机ip
        }
}
