简介

  • 客户端把sessionId保存在HTTP头中

  • HTTP头需要设置gzip压缩

  • 服务器把errorCode, errorMsg保存在HTTP头中。客户端每次请求都要检查http头中是否返回errorCode和errorMsg数据，如果有值，则弹出errorMsg消息，终止以下逻辑执行。errorCode为1，结果返回成功

  • 客户端在每个json请求中，都需要传入1.0 平台信息对象 platformInfo和用户token。
